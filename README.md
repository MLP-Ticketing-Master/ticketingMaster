## 🖥️ 프로젝트 소개

티켓팅마스터는 사용자가 다양한 이벤트의 티켓을 조회하고 예매할 수 있는 서비스입니다.  
대용량 트래픽 상황에서도 안정적인 예매 처리를 목표로 설계되었으며, 부하 테스트를 통해 성능을 검증하였습니다.

| 구분 | 내용 |
|------|------|
| 프로젝트 기간 | 2025.xx.xx ~ 2025.xx.xx |
| 팀 구성 | 백엔드 x명, 프론트엔드 x명 |
| 배포 URL | - |

## ✨ 주요 기능

### 👤 사용자
- 회원가입 / 로그인 (JWT 기반 인증)
- 이벤트 목록 조회 및 검색
- 티켓 예매 및 결제
- 예매 내역 조회 / 취소

### 🛠️ 관리자
- 이벤트 등록 / 수정 / 삭제
- 예매 현황 조회

### ⚡ 성능
- 동시 다수 요청 처리를 고려한 예매 로직 설계
- 부하 테스트(loadtest)를 통한 성능 검증
